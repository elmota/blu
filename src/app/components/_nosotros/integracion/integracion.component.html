Integración Web Services

Servicios de Integración Blue Express

Blue Express tiene disponible para sus clientes servicios de integración mediante tecnología SOA (Arquitectura Orientada a Servicios).

¿Qué es un Web Services?

Es una tecnología estandarizada que nos permite intercambiar datos entre distintas aplicaciones que pueden estar desarrolladas en lenguajes de programación diferentes.

Ventajas de un Web Services

Principalmente nos aporta interoperabilidad entre aplicaciones  independiente de la plataforma sobre las que se instalen.

Disponibilidad

Los servicios se encuentran disponible 7x24  siendo monitoreados por el área de IT de Blue Express.

Requisitos Cliente

  1. Entregar IP / HOST desde donde se realizaran las integraciones para agregar estos datos a la lista blanca de accesos al Web Services.
  2. Poseer una cuenta corriente vigente con Blue Express.
  3. Poseer un sistema que permita realizar integraciones con tecnología SOA (Arquitectura Orientada a Servicios) o desarrollos internos realizados en Java, Visual Estudio, PHP, Phyton o cualquier otro lenguaje que permita programación orientada a objetos/aspectos.
